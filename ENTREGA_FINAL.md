## 🚀 Cómo Usar

### Desarrollo

```bash
npm install
npm run dev
# → http://localhost:3000
```

### Producción

```bash
npm run build
# Sitio generado en /out
```

### Deploy

Subir la carpeta `/out` a:

- GitHub Pages
- Netlify
- Vercel
- Hosting tradicional (FTP)

Ver guía completa: `documentacion/DEPLOYMENT.md`
